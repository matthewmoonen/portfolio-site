An Open Source Mouse Configuration tool for Linux - View on GitHub

Featuring customisable buttons and gesture controls, the Logitech MX and M series mice are popular productivity tools amongst professionals and programmers

There was no Logitech software for Linux, so I made my own and released it as an open-source project.

The MX and M series have substantial mature apps developed by Logitech, but the company themselves do not make a version of their software for Linux. In 2020, a community-developed driver was released which brought most of the configurability of these mice to Linux, but there was a problem. Many users found these drivers tedious and confusing to set up, requiring a complicated configuration file and a lot of reading and research on the user's part to decifer There had been discussions on GitHub for several years about developing a GUI (see here), but no project had proceeded past the early requirements gathering stage. I decided that needed to change.
